# Математические рассуждения в LLM
В этом проекте реализован код, обучающий большие языковые модели (LLM) математическим рассуждениям; основная техника — Supervised Finetuning (SFT), при котором модель учится на ответах, сгенерированных DeepSeek-R1, перенимая из них сложные навыки математических рассуждений (рефлексия, перепроверка вычислений и т. д.).
Этот эксперимент основан на датасете [DeepMath-103K](https://huggingface.co/datasets/zwhe99/DeepMath-103K) и модели [Qwen/Qwen2.5-Math-1.5B](https://huggingface.co/Qwen/Qwen2.5-Math-1.5B) и проводит читателя через полный цикл дистилляции данных на математических задачах для первого знакомства.

## Цель эксперимента
- Познакомиться с очисткой и предобработкой математических датасетов
- Познакомиться с базовыми методами дообучения больших моделей
- Познакомиться с базовыми методами инференса с помощью больших моделей
- Познакомиться с базовыми методами оценки математических способностей модели

## Требования к окружению

- Python 3.8+
- PyTorch
- Transformers
- Datasets
- vllm
- GPU с минимум 40 ГБ видеопамяти

## Документация эксперимента

1. Откройте Jupyter notebook:
```bash
jupyter notebook sft_math.ipynb
```

2. Notebook содержит несколько основных частей:
   - Скачивание и предобработка датасета
   - Загрузка и обучение модели
   - Генерация моделью и оценка

3. Конкретные действия и пояснения см. в notebook.

## Выходные файлы

В процессе обучения создаются файлы чекпойнтов обучения, а после завершения генерации сохраняются результаты генерации.

## Важные замечания

- Модель и данные нужно загружать с huggingface; в Китае можно скачивать через зеркало https://hf-mirror.com.
- Убедитесь, что есть достаточно места на диске для хранения файлов исходной модели, файлов данных и чекпойнтов после обучения (зарезервируйте минимум 50 ГБ места)
- Требуется GPU с минимум 40 ГБ видеопамяти
